## Get started

1. Install dependencies

    ```bash
    npm install
    ```

2. Run Fastify API

    ```bash
     npm run api
    ```
3. Seed some data (a user for server context)

    ```bash
     npm run data-model-postgres
    ```

3. Run Client
   For Web (easy for testing)  run:

```bash
 npm run web: dev
```
  For mobile(Expo): 
  ```bash
 npm run start
```
